Objects with behaviour

Represent anything with dynamic objects. Define behaviours to make them alive. OBJY will do the rest.

Model anything, simply

Use standard JSON to model any object with simple attributes and time or event-driven behaviours.

    valid: true,
    onChange: "if(!this.valid) email(...)"
}).add()
